Estilos CSS3 Externos

É o a melhor técnica de aplicação, não importa quantas páginas existam no seu site, somente com um arquivo de css você muda para todos. Vamos criar duas páginas agora e modificalas com um arquivo CSS3 externo. Para criar ele vá em HEAD e coloque link:css, no href que aparece coloque o mouse em cima, vai aparecer um popup com os dizeres "siga o link", clique e vai aparecer uma mensagem de erro dizendo que o arquivo não foi encontrado com a opção de criar arquivo, clique em criar arquivo e pronto o arquivo css3 foi criado com o nome de "style.css". No arquivo css é bom usar regras (as regras são tipo aas configurações de compatibilidades que vemos na parte Head do html, so que em css são chamadas de regras), colocamos a regra chamada @charset "UTF-8";. Tem que ser escrita assim, toda a regra em css começa com @ e é colocada no inicio do arquivo css.

Perceba que não precisamos mais na rag style quando estamos no aqruivo css, basta somente ir colocando os sletores. Para a segunda página ficar tbm sobre o efeito do arqueivo css é preciso colocar na parte ghead dela tbm o link:css.

Observação

Você pode utilizar mais de um link:css no head, por exemplo para estilos extras que só algumas páginas tem

Evite ao máximo usar a técnica inline, só use se for fazer uma alteração pontual que só serve naquela página naquela tag especifica. Você até pode misturar as técnicas mas faça isso com sabedoria.
