Estudo sobre Links

Links Externos

Vai apontar para outro site que não é nosso, usamos a tag "A"(âncora) dentro da tag vai ter o parâmetro href(referência a hipertexto, pra onde o site vai), nele tu coloca a url do site.

Ex: Você pode acessar o repositório do Github

Ex:Você pode acessar também o meu canal do youtube.

Repare que ao clicar no link externo, a sua página some, para evitar isso use os parâmentros: target="_blank"(Que faz a página abrir em outra guia) e rel="external". Observe o que acontece agora: repositório do Github. Uma nova aba se abre e sua página se mantêm.

Então, se você vai abrir um site que não é seu, é recomendado que você abrar em outra aba.

Links Internos

Link que faz uma ligação com uma página dentro do nosso servidor e diz repeito a nossa própria página.

Dentro do arquivo HTML você cria uma segunda página. Usamos a tag "A" nesse caso também, dentro do parâmetro href aperte ctrl+space e selecione a página.

Vamos fazer um link para ela.

Segunda página

OBS:Nos links externos usamos o parâmetro rel="external", já nos links internos, na página principal usamos rel="next"(próximo) pq estamos dizendo que é um link para a proxima pagina e na segunda página usamos o rel="prev"(anterior), pq estamos dizendo que é um link para voltar para alguma página anterior. Isso ajuda o mecansimo de busca a encontrar sua página com mais facilidade.

OBS: Nos links internos não usamos a target="_blank" pois não queremos abrir oura guia no navegador, queremos abrir ma propria guia aberta a nossa próxima página, então usamos o target= "_self" para indicar para o google que a página vai abrir na prória aba.

Links para Downloads

  • Baixar o livro em PDF
  • Livro compactado em ZIP

Para baixar você coloca dois parâmentros: Download="nome do arquivo"(para sinalizar que é um arquivo para baixar) e type="application/pdf"(tem varios applications para casos especificos).

A depender do navegador o arquivo em pdf não vai baixar e sim abrir na guia, mas se clicar com o botão direito e clikar em salvar como ele salva o no seu computador.

OBS:Quando não tiver arquivo coloque no href um # para simbolizar que o arquivo está vazio.

OBS:É bom transformar o arquivo em ZIP pq diminui o tamanho dele.

OBS:Site iana.org vai dar todas os type media necessários para pedir para o google baixar o arquivo do link criado.
