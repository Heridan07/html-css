Código Fonte

Conceito:"O código-fonte de um software é um conjunto de arquivos de texto contendo todas as instruções que devem ser executadas, expressas de forma ordenada numa linguagem de programação. Essas instruções determinam o que um programa de computador deve fazer – o que ele deve apresentar e como ele deve se comportar."

Ex: O comando document.getElementById é escrito em lingugam Java Script, repare que o comando fica difícil de ler quado misturamos letras minúsculas e maiúsculas por isso vamos usar uma tag especifica para código fonte, a tag "CODE" que deixa ele assim: document.getElementById. Com fonte monoespaçada que é boa para ler códigos e é isso que essa tag faz.

Tag "PRE"

EX:

num= int(input('Digite um número')) if num % 2 == 0: print(f'O número {num} é Par') else: print(f'O número é Ímpar') print('Fim do programa')

Repare que criamos um código python e colocamos ele dentro de uma tag CODE, mas essa tag não trabalha com identação, logo fica tudo como se estivesse dentro de um parágrafo, para que tudo fquei organizado, existe a tag chamada PRE, é só colocar a tag CODE dentro da PRE que tudo fica organizado. Tudo dentro da Tag PRE é considerado, as quebras de linha e espaços, é melhor do que usar a tag BR.

EX:
