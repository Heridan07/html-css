Listas OL e UL

Listas são muito utilizadas pricipalmente nos menus de sites.

Listas OL (Listas Ordenadas)

São aquelas que a ordem importa, a tag é <OL>, antes de escrever os itens tu vai usar a tag LI (list), que vai estar dentro da tag OL, é na tag LI que você coloca os itens, pra cada item você vai fazer uma tag LI.

  1. Acordar
  2. Arrumar a cama
  3. Tomar café
  4. Escovar os dentes

Personalizando listas ordenadas

Podemos personalizar as listas colocando o parâmetro type, que nos dará cinco opçoes de formatação (a primeira já vimos, é a padrão):

2- A (Ordenadas por letras Maiúculas, a outra é o "a" minúsculo)

Receita de chá
  1. Ligue o fogo
  2. Espere a água ferver
  3. Ao ferver apague
  4. Jogue as folhas
  5. Tampe
  6. Deixe em efusão por 10min

4- I (algarismos romanos, o último é o "i" minúsculo)

Abdominais
  1. Deite-se
  2. Mãos atrás da cabeça
  3. Contraía do músculos abdominais na subida enquando expira
  4. Descontraia controladamente na descida enquanto inspira

Você também pode querer que a lista começe de uma ordem especifica para isso você tem o parâmetro start="". EX:

  1. Acordar
  2. Arrumar a cama
  3. Tomar café
  4. Escovar os dentes

OBS: O start só lê números então se quiser que, por exmplo o Type="A" Começe pelo B, voce tem que colocar assim: Type="A" Start=2.
