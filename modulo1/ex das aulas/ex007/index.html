Semântica e Tags Absoletas

O HTML5 e suas tags são focadas na semântica(significado) e não na forma (Este é o trabalho de CSS), algumas tags do html já se tornaram absoletas em consequência disso(pois antigas versões do HTML não eram focados totalmente em semântica), antigamente para colocar um endereço utilizava-se a tag <u> hoje em dia se utiliza somente a <address> por ser mais semântica. Se utilizar uma tag absoleta pode até funcionar mas o google pode não indicar muito o seu site justamente por estar utilizando elements ultrapassados. Para saber se uma tag está absoleta basta pesquisar ela no site "absolete features".
