Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag "i" (não semântica).

Nesta temos um termo em ênfase usando a tag "em" (semântica) que significa ênfase.

Observações

Primeira

Esqueça as palavras ítalico e negrito quando estamos em HTML, o correto será falar ênfase e destaque, ítalico e negrito remetem mais a forma e isso nós vamos ver só em CSS.

Segunda

Atalho no VScode (editor que estamos usando para fazer o arquivo HTML): Se quiser envelopar palavras ou frazes nas tags selecione elas com o mouse, depois utilize ctrl+shift+P, procure wrap with abreviation, pressione enter, digite a tag que quer e pressione enter novamente, assim você vai envelopar a tag mais rapidamente do que ficar arrastando as frases/palavras para dentro das tags.

Texto marcado

Podemos criar um texto marcado usando a tag Mark

A cor padrão é amarelo, em HTML você não tem uma maneira de mudar a cor, você vai ter que usar CSS dentro de html através do parâmetro Style. Ex: mark style="background-color:blue;"

Mas se você quiser colocar outra palavra marcada você vai ter a cor amarela padrão, para mudar isso você vai em HEAD e vai usar a tag style(uma tag css) dentro da tag style vai usar o usar o seletor mark e colocar a cor que quer, isso vai mudara cor padrão da tag mark, no caso coloquei a cor padrão para laranja

Texto grande e pequeno

Estamos criando um texto grande com a tag BIG e um texto pequeno com a tag SMALL

A tag BIG se tornou absoleta pois não dá para atribuir sgnificado, já SMALL ainda se mantêm, pois há contratos que por exemplo pedem para que você leia as letras pequenas, então ainda há sentido em usar ela(É um palpite).

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado, usando a tag DEL.

Texto sobrescrito

Pra inserir coisas do tipo x20+3(x elevado a 20), usamos a tag SUP. Fica assim: x20+3.

Texto subscrito

Para escrever coisas do tipo H2O(no caso o dois fica mais para baixo), usamos a tag sub. Fica assim: H2O.
