Estilos Locais / Internos de Css

É quando usamos css na parte Head do HTML, trabalhos no head na forma que realmente se trabalha em css. Usamos os seletores que tem o nome da tag que queremos modificar mudamos a cor desse site assim:

        <head>
            <meta charset="UTF-8">
            <meta http-equiv="X-UA-Compatible" content="IE=edge">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <link rel="shortcut icon" href="favicon.curso.ico" type="image/x-icon">
            <title>Estilos locais/Internos</title>
            
                <style>
                    body{
                        background-color: rgb(129, 212, 212);
                    }
                </style>
            
        </head>

Repare que a tag style, dentro dela usamos um seletor body, que tem o mesmo nome da tag que queremos modificar, abrimos chaves e dentro dela vamos colocar as declarações que são o connjunto de propriedade(no caso é background-color) e valor(rgb 171,212,212) e no final colocamos ";". podemos usar vários seletores dentro de uma mesma tag style, só colocar um embaixo do outro, se colocar que a tag h1 vai ser vermelha todas ficam sem precisar ir de tag em tag.

Observações

Se você está fazendo uma página simples que precisa de mudanças locais use essa técnica, mas se tiver muitas páginas vamos usar os estilos externos

Desvantagens

O estilo ocupa mais espaço que o conteúdo, então você vai abrir o arquivo para mudar algo no body por exemplo e aí tu vai que descer tudo para conseguir mexer na corpo. E se tiver muitas páginas vai ter qu e copiar e colar o código, se tiver que mudar um seletor vai ter que ir de página em página. Logo, essa técnica serve para quando se tem só uma página.
