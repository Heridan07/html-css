Vídeos

Sobre vídeos no HTML5

No site Pixels você consegue baixar imagens e vídeos gratuitos.

Os formatos suportados pelos navegadores geralmente são: MP4,OGV, M4V e WEBM.

Um dica para diminuir vídeos grandes. Use o programa HandBrake ele pode mudar o formato dos seus vídeos(MP4,OGV(esse é o único que não está disponível), M4V ou WEBM) tbm e adequar ele para ser utilizado em Web. Temos que criar vídeos em vários formatos pois navegadores tem compatibilidades diferentis com formatos de arquivo diferentes.

Cuidado com os Custos:

Se você colocar uma qualidade muito grande da imagem, o plano de hospedagem mensal fica mais caro, se o cliente pedir hospensagem local piora a situação, pq a conta vai pra tu acho. Acima de 1080p não hospede nada no seu site.

Inserindo vídeos

Crie a tag video e depois coloque as tags source dentro, o resto segue os principios de inserir audio.

Use os parêmetros Width="500"(É a largura d vídeo, coloque em 500 pois é um bom tamanho, mas tu que escolhe), poster=""(Nessa você usa uma imagem para servir de capa para o vídeo, mas tem que ser do mesmo tamanho se não fica um negócio doido kkkkk) e controls, pode usar parâmetros como loop e autoplay tbm.

Cuidado! custo de hospedagem local

Nós colocamos Vários sourcescom vários formatos de arquivo como podemos ver:

        <video width="500" poster="imagens/foto-p.png" controls>
            <source src="videos/pexels-darlene-alderson-7040123.mp4" type="video/mp4">
            <source src="videos/pexels-darlene-alderson-7040123.webm" type="video/webm">
            <source src="videos/pexels-darlene-alderson-7040123.m4v" type="video/m4v">
            <source src="videos/pexels-darlene-alderson-7040123.ogv" type="video/ogv">
            <p>Seu navegador não tem compatibilidade com reprodução de vídeo.</p%gt;
        </video>

Isso é bom pois permite que mais aparelhos tenham compatibilidade, mas é ruim também pois encaresse o valor da hospedagem a depender do arquivo que seja utilizado, por exemplo o fomato Ogv está muito pesado, se muitas pessoas acessarem um dispositivo que é compativel só com esse formato teremos um problema, se o cliente quiser que o vídeo fique hospedado no site dele vai ficar caro para ele, se hospedar com você vai ficar caro pra tu

Incorporação de vídeos externos

Do youtube

Vai no vídeo que quer clica em compartilhar, vai em incorporar, ele vai montar um código html e aí tu copia e cola

Quando o vídeo é do youtube consome muito pouco, ou nada, então o custo é menor. então o recomendado é que seu cliente tenha um canal no youtube e poste os vídeos lá.
