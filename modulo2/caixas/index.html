Box Model

Todo elemento visual dentro de uma página é uma caixa, um parágrafo um título uma div etc.

Elementos

Medidas: altura= height, largura = width.

Estrutura:

conteúdo -> padding (espaço entre o conteúdo e a borda) -> border -> margem (Espaço entre a borda de uma caixa e algum outro elemento)-> outline (entre a borda e a margem).

Tipos de caixa:

Box-level

Sempre inicia na proxima linha e ocupa a largura total da tela (é sempre 100 por cento), ex:

  1. div
  2. h1-h6
  3. main
  4. header
  5. nav
  6. article
  7. aside
  8. footer
  9. form
  10. video

Inline-level

Pula para o lado da linha, não a quebra, ele fica dentro dela, não ocupa o tamnaho inteiro da tela, ex:

  1. span
  2. a
  3. code
  4. small
  5. strong
  6. em
  7. sup e sub
  8. label
  9. button
  10. input
  11. select

Formas de utilizar o padding:

1ª)

                padding:10px;

Todos s lados ficam com padding 10px.

2ª)

                paddind top:10px;
                padding-right:10px;
                padding-bottom:10px;
                padding-left:15px;

Todos os lados são configurados individualmente, tem que ser configurados no sentido horário.

3ª)

                padding:10px 15px 7px 8px;

Um shorthand cada configuração segue a ordem top - right- bottom e left.

Obs: Essas relações também se aplicam a marin, border e outline.

Sobre a margem

Sempre que quiser centralizar uma caixa ou uma div é só colocar "margin:auto;"

se quiser centralizar só a largura coloque por exemplo: "margin: 10px auto 20px auto;".

Outline

Exemplo

Esse tracejado em azul é o outline, que está circundando a borda, o outiline fica dentro da margem e pode ultrapassar os limites dela dentro de uma div, mas não aumenta nem diminui a margem.

OBSERVAÇÃO:

No dev tolls as caixas d tipo box-level são chamadas de display:block e as inline level de display:inline. Se quiser trasformar uma caixa box level para inline level é só fazer isso:

                h1{
                    display:inline;
                }

Trasnformamos o h1 que é box-level em inline-level através da declaração display, quando realizamos essas trasnformação não podemos mais alterar o width o height.

Podemos fazer o inverso também colocoando: display:block;

Podemos fazer com que uma caixa inline-level também adiquira características de box level através da declaração: display:inline-block, assim você pode mecher no tamanho, altura e largura.
