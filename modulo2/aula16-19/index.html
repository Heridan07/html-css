Aula Dezesseis á Dezenove

1ª) Criar gradiente de cor:

background-image: linear-gradient(to top,rgb(252, 214, 220),rgb(215, 174, 215));

Talvez seja necessário criar um seletor universal com essa declaração:

        *{

            height: 100%;

        }

Dessa forma se o degrade for de cima para baixo o site não ficará zebrado, mas talvez esse metodo não funcione, caso isso ocorra use o background-attachment: fixed;: 

                main{

                    background-image: linear-gradient(to top,rgb(252, 214, 220),rgb(215, 174, 215));
                    background-attachment: fixed;
                    
                }

2ª) Sombra na caixa:

            box-shadow: 5px 5px 10px , blue;

O primeiro parametro é quanto você quer que a sombra vai andar para direita, o segundo é quanto ela vai para baixo

o terceiro é quanto ela vai se espalhar, o último é a transparência e a cor dela.

3ª)Peso da fonte:

font-weight:;

Podem ser literais: lighter, normal,bold e bolder.

Podem ser númericas: vão de 100 á 900.

OBS: Nem todas as fontes suportam todos os tipos de peso.

4ª) Propriedade Shorthand:

Quando queremos resumir um conjunto configurações uma única linha, para isso é necessario que pertençam ao mesmo tipo de configuração, ex configurações relacionadas a font:

            main{
                font-family: 'Work Sans' sans-serif;
                font-weight: bolder;
                font-size: 3em;
                font-styke:italic;
            }

Para colocar em uma única linha as confiurações de fonte é necessário seguir uma ordem de precedência: primeiro colocamos o style depois vem weight, size e family, ex:

font:italic bolder 3em 'Work Sans', 'sans-serif';

5ª) Alinhamento de texto:

Alinhar a esquerda ou direita

text-align:left;

Centralizar

text-align:center;

Justificar

text-align:justify;

Recuo de paragráfo:

text-indent:20px;
