Git & Github

Uma introdução ao gerenciamento e repositório de arquivos

Repositórios

Existem dois tipos, aqueles criados no nosso próprio computador chamados de Repositórios locais e os repositórios remotos criados online, onde podemos fazer backup dos nossos arquivos.

Gerenciador de versões e repositório local

É um software que pode ser instalado no computador, com a função de guardar as versões do arquivo qque você solicitar, assim poderá voltar a versão anterior se a nova não for adequada aos seus objetivos. Um exemplo de software gerenciador de versões é o Git, este gerenciador é gratuito e foi criado pela mente por trás do servidor Linux. Em suma, o Git é o repositório local.

repositório Remoto

Importante destacar que o Git somente salva as versões dentro do seu computador, logo é importante a existência de um software que guarde esses arquivos na nuvem, um exemplo é o GitHub, utilizado para fazer backup gratuito de seus projetos, é um servidor remoto. Em suma. o GitHub é o repositório remoto.

O push

O GitHub permite trabalhar em equipe, compartilhando o código, os membros da equipe podem mexer o no código localmente (na máquina deles) e atualizar o mesmo repositório remoto (isso se chama de push).
